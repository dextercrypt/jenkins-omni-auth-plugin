When enabled, Jenkins fetches the user's Azure AD group memberships on every login and makes them available to Jenkins' authorization strategies (e.g. Matrix-based security).

Requirements:
  • Add GroupMember.Read.All as a delegated API permission on your app registration
  • Grant admin consent for that permission in Azure Portal
Group names in Jenkins must match the display name of the group exactly as it appears in Azure AD.

Leave unchecked if you only need single sign-on without group-based permissions.
